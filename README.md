# Retail-Sales-Forecasting-using-Time-Series-Analysis
This project analyzes retail sales data using time-series techniques and builds a forecasting model to predict future sales trends.


## 🚀 Key Features
- Data Cleaning & Preprocessing  
- Time-Series Analysis  
- Trend & Seasonality Detection  
- Data Visualization  
- Forecasting using Linear Regression  

## 📊 Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## 📈 Outcome
The project identifies key sales trends and predicts future sales, helping in business decision-making such as inventory planning and marketing strategies.
